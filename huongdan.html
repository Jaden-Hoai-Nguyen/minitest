📝 Bước 1: Cấu trúc Google Form & Sheets

Google Form của bạn đã được tạo. Khi học sinh submit form, dữ liệu tự động vào Google Sheets.

Tuy nhiên, để website đọc được câu hỏi, bạn cần tạo một sheet riêng với cấu trúc sau:

Cột (Header)Ý nghĩaVí dụ
QuestionCâu hỏi2x + 6 = 0, nghiệm x = ?
TypeLoại câumcq | fill | dragdrop | matching
OptionsCác đáp án (ngăn cách |)x=3 | x=-3 | x=6 | x=-6
CorrectĐáp án đúngx=-3
ImageURL hình ảnh (nếu có)https://...
💡 Tip: Tạo Sheet riêng tên "Questions" trong cùng Spreadsheet. Row 1 là header, từ row 2 trở đi là câu hỏi.

🔗 Bước 2: Publish Google Sheets thành JSON

1

Mở Google Sheets của bạn

Mở file Sheets liên kết với Google Form (Test 1_K7, Test 2_K7...)

2

Lấy Spreadsheet ID

Từ URL của Sheets, copy phần ID:

https://docs.google.com/spreadsheets/d/[SPREADSHEET_ID]/edit ↑ Copy phần này
3

Tạo URL JSON (Phương pháp 1 – GViz API)

Dùng Google Visualization API (không cần publish):

https://docs.google.com/spreadsheets/d/[ID]/gviz/tq?tqx=out:json&sheet=Questions

Thay [ID] bằng Spreadsheet ID của bạn.

⚠️ Lưu ý: Sheet phải được chia sẻ "Anyone with the link can view"
4

Publish to web (Phương pháp 2)

File → Share → Publish to web → Chọn sheet "Questions" → CSV → Publish

URL dạng: https://docs.google.com/spreadsheets/d/[ID]/export?format=csv&gid=0

⚙️ Bước 3: Nhập URL vào MiniTest

1

Mở MiniTest website

Mở file index.html trong trình duyệt

2

Mở phần cấu hình

Bấm vào "⚙️ Cấu hình Google Sheets URL" ở cuối trang chủ

3

Nhập URL cho từng test

Dán URL JSON của từng Test vào ô tương ứng → Bấm "💾 Lưu cấu hình"

💡 URL được lưu vào localStorage – Chỉ cần nhập 1 lần, lần sau tự động nhớ.

🚀 Bước 5: Chạy website

Cách 1 – Mở trực tiếp: Double-click index.html

Cách 2 – Dùng Live Server (VS Code):

npx live-server .

Cách 3 – Deploy lên GitHub Pages / Netlify để học sinh dùng online

⚠️ CORS: Nếu fetch bị lỗi CORS, deploy website lên server thật hoặc dùng GitHub Pages. Local file:// có thể bị chặn.
